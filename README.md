# 🤖 AI Resume Analyzer

An AI-powered resume analysis platform that helps users evaluate resumes, improve ATS compatibility, and receive intelligent feedback using modern AI technologies and scalable full-stack architecture.

# ⚙️ Installation & Setup

## Clone the repository

```bash
git clone https://github.com/siddeshrizwani/AI-Resume-Analyzer.git
```

## Navigate to project directory

```bash
cd AI-Resume-Analyzer
```

## Install dependencies

```bash
npm install
```

## Run the development server

```bash
npm run dev
```
